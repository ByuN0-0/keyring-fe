# Keyring Frontend

Next.js frontend for the Keyring app.

## Requirements

- Node.js 18+

## Setup

```bash
npm install
```

## Environment

Create `keyring-fe/.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8787
```

## Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
